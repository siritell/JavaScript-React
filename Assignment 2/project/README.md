# Hey Siri

A restaurant review blog built with React, Node.js, Express and MongoDB.

## Live Demo

[heysiri-blog.vercel.app](https://heysiri-blog.vercel.app)

## About

Hey Siri is a full-stack restaurant review app that fetches and displays reviews from a MongoDB database. Each review includes the restaurant name, address, a description and standout dishes.

## Tech Stack

- **Frontend:** React, deployed on Vercel
- **Backend:** Node.js, Express, deployed on Render
- **Database:** MongoDB Atlas

## Features

- Fetch and display restaurant reviews from a live database
- Full CRUD functionality implemented (add, edit, delete currently disabled in public version)
- Loading spinner while data is being fetched
- Responsive design

## Notes

The backend is hosted on Render's free tier and may take 30–60 seconds to wake up on the first load.
